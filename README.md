# 🛠️ JS Mini Projects

A curated collection of mini JavaScript projects to strengthen core concepts, enhance DOM manipulation skills, and build practical web development experience.
Each project is standalone and focuses on a specific functionality or use-case.


### 📁 Projects Included

#### 🎮 Snake Game
A classic Snake game built using HTML, CSS, and vanilla JavaScript.

#### ⏰ Digital Clock
A real-time digital clock that updates every second, displaying the current time.

#### 🌦️ Weather WebApp
A simple weather app that fetches and displays real-time weather data using a public API.

#### 🎹 Virtual Keyboard
An interactive on-screen keyboard that simulates typing and keypress feedback.

#### 📋 Student Details
A form-based app to input, validate, and display student information dynamically.


### 📦 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Java & JDBC


### 🚀 How to Run

1. Clone the repository
    ```sh
    git clone https://github.com/your-username/js-mini-projects.git 
    cd js-mini-projects
    ```

2. Open any project folder
3. Launch index.html in your browser
