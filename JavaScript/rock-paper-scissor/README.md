# ✊✋✌️ Rock Paper Scissors Game (JavaScript)

A simple and fun **Rock Paper Scissors** game built using **Vanilla JavaScript**, HTML, and CSS. This project was created for practicing frontend logic, DOM manipulation, and handling user input events.

### 🧰 Tech Stack
- HTML
- CSS
- JavaScript

### 🎯 Features
- Play against the computer
- Random computer choices
- Real-time result display
- Score tracking (optional)
- Responsive design (if included)

### 🚀 How to Run

1. Clone or download the repository
2. Open the `index.html` file in any browser

```bash
open index.html
```
