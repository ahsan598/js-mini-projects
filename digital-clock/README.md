# 🕒 Digital Clock (Java Swing)

A simple digital clock built using **Java Swing**. It displays the current system time in real-time and is designed for practice purposes, especially for learning GUI components in Java.

### 🧰 Tech Stack
- Java
- Java Swing

### 🎯 Features
- Real-time digital clock
- Lightweight and responsive window

### 📸 Screenshot

### 🚀 How to Run

1. Open the project in any Java IDE
2. Run the `Main.java` or `DigitalClock.java` file

```bash
javac DigitalClock.java
java DigitalClock
```
