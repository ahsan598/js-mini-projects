# 🎮 Java Snake Game

A simple Snake Game built in Java using Swing for UI and multithreading for the game loop. This was developed as a practice project to improve my understanding of GUI-based Java applications.

### 🧰 Tech Stack
- Java
- Java Swing

### 🎯 Features
- Classic snake movement using arrow keys
- Game over screen
- Score tracking
- Thread-based game loop

### 📸 Screenshot
![snake-game](https://github.com/ahsan598/js-mini-projects/blob/master/snake-game/screenshots/snake-game.png)

### 🚀 How to Run
1. Clone the repository
2. Open the project in any Java IDE (like IntelliJ or Eclipse)
3. Compile and run the `Main.java` file

```sh
javac SnakeGame.java
java SnakeGame
```
